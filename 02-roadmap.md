# 02 – Now-Next-Future Roadmap

## 💡 Why this exercise matters
A roadmap keeps the team realistic and communicates trade-offs clearly.

## 🎯 Expected Outcomes
- Sequenced plan for 3 weeks (NOW) plus future ideas  
- Visible alignment on priorities and dependencies  

---

### 🔧 Prompt – Create Your Roadmap
**NOW (Weeks 1–3)** → MVP build that proves core hypothesis  
**NEXT (Month 2)** → Enhancements for trust / scalability  
**FUTURE (Month 3 +)** → Expansion or automation features  

**Example:**  
- NOW – Guided Intake + Matching API + Results UI + Feedback  
- NEXT – Live job feed integration + advanced analytics  
- FUTURE – Personalized upskilling and career tracking
