# 00 – MVP Scoping

## 💡 Why this exercise matters
You now have validated problems, prototypes, and early user insights.  
The next step is deciding **what can be built in 3 weeks by 3 people** — enough to prove value and learn fast.

Adapt prompt to your team composition.

## 🎯 Expected Outcomes
- Realistic MVP definition tied to your hypothesis  
- List of must-have vs nice-to-have features  
- Alignment on team learning goals  


 
---

### 🔧 Prompt – Define Your MVP
**Team Context**  
Team size: 3 
Time: 3 weeks 
Core skills: [data / AI / frontend / PM]

**1️⃣ MVP Goal (user + learning focused)**  
Example:  
> Build a working prototype that turns a user’s project story into three career matches, each with an AI-generated “why this fits” explanation.

**2️⃣ Core Hypothesis to Validate**  
Example:  
> If users tell one project story + a few motivation cues, they’ll trust AI-based recommendations more than keyword matches.

**3️⃣ MVP Core Components (must-haves)**  
- Feature 1: Guided intake (flow + API)  
- Feature 2: Model / matching logic  
- Feature 3: Results + feedback UI  

**4️⃣ Later Features**  
List *Should Have* and *Nice to Have* for future iterations.

**5️⃣ Individual Learning Goals (optional)**  
- Person A – Backend / AI Dev goal = …  
- Person B – Data Engineer goal = …  
- Person C – PM / Designer goal = …  

✅ Deliverable: One-paragraph MVP summary + feature list + learning focus.


---

## Alternative prompt


You have: validated prototype + user insights + clear metrics.  
Your team: 3 people • 3 weeks • skilled in Python, model deployment & AI tools (e.g., Lovable for frontend).



Based on our prototype and user insights:

1. What would be the MVP of this product that we can build and test in 3 weeks?  
   (Focus on what validates the product’s value, not every feature.)
2. What hypothesis does this MVP help us confirm or reject?
3. List the core components (must-have features) and which are optional.
4. Considering our technical strengths, what stack or workflow should we use?
5. What is each member’s personal learning goal as part of this MVP?
